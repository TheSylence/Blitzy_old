Blitzy manual

Welcome to the Blitzy manual. In this document you will find every information that you need to know about Blitzy in order to use it and even more. This manual was written for Blitzy 1.0.0. If you are viewing this document online please make sure that you have the latest version of Blitzy installed.

The manual written for the version you have installed is located in the "docs" folder of your Blitzy installation. To make sure the information is correct please refer to the document that came with the version you installed.

System requirements

To use Blitzy you will need:

  • Windows Vista or newer (64bit) with the latest Service Pack installed
  • .NET Framework 4.5.1
  • Some free space on one of your hard disks
  • Some free space on your system disk (*)

Please don't ask me for other specs like CPU speed or RAM. To be honest, I don't know them and I don't have enough computers to test this. However, if your computer can run Windows it probably is also able to run Blitzy.

Oh and yes I do not support Windows XP! If you managed to get Blitzy to run under XP that's good for you. However don't expect any support if you encounter problems that are specific to XP.

(*): Blitzy stores its settings and the catalog in the appdata folder. By default this folder is stored on your primary hard disk (the one Windows is installed on). The exact amount of space needed depends on how big your catalog is and what plugins you have installed. However I doubt you will need more than 20 or 30 MB of space for your catalog and settings.

Installation

Installing Blitzy is pretty straight forward. After downloading the setup file from the website all you have to do is double clicking the file. Then the setup wizard will open and guide you through the installation process. After the wizard finished you may start using Blitzy.

Using Blitzy

Blitzy was designed to be as easy and fast to use as possible. And if you used other keystroke launchers in the past, working with Blitzy should be as easy as eating a pancake.

To bring up Blitzy's window you have to press the shortcut you defined in the settings. (The default shortcut is Ctrl+Alt+Space) You can do this from anywhere and not just from the desktop, browser, or your mail client. However be sure to use a shortcut that will not interfer with shortcuts used by applications you try to open Blitzy from. Also it is recommended to use a shortcut that is not used globally by other applications.

Entering commands

To enter a command you simply start typing. Blitzy will display a list of suggestions from which you can choose using the cursor keys (up and down). By pressing the return key Blitzy will execute the command that is currently selected. Additionally you may execute a command from the list by double clicking the desired one.

Tip: You don't need to type in the whole command. When the item you want to execute is selected you can simply press the return key and the command will be executed, independent of what is currently typed into the input box. You may even have some spelling errors. The only thing that matters is that the command you want to execute is selected.

Some commands may require or offer additional parameters to be entered. To do so enter the command and press the tabulator key. Then simply type as many parameters as the command needs (each separated with another tabulator) and press the return key when you are done. Each parameter will be separated by an arrow and most of the commands should indicate if they need more parameters and what kind of parameters they require.

Note: Commands are executed in the background. This means, that depending on how much time the execution of the command takes, you may have to wait until you see the action. However this has the advantage that you can execute a new command right after the previous command was started. You don't have to wait until the previous command finishes.

Blitzy has the ability to learn what commands you use often and places items you use more often on top of the suggestion list. So the longer you use Blitzy, the less you have to type since the command you want to execute is already placed on top!

Executing commands with elevated rights

Blitzy has the ability to execute commands with elevated rights (a.k.a. as an administrator). To do so you need to have access to the account of an administrator on your local machine.

If you want to execute a command with elevated rights you simply need to type in the command as you would normally do and select the right command from the command list. After that, hold Ctrl and Shift while pressing return. Whatever command you select will be executed with elevated rights.

Note: Executing some commands with elevated is useless and therefor disabled. For example: There is no benefit in rebuilding the catalog with administrative rights. In fact, the UAC prompt (if enabled) won't even pop up for certain commands.

Predefined commands

There are some commands built into Blitzy that can be used no matter what plugins are installed or not. Below you can find a list of all these commands:

Command Description
quit Exits the application
reset Resets the command execution count. After executing this command the order in which commands are listed when you type is reset to default values. This can be useful if you messed up the counting, don't longer use a command you previously used or you simply want to start over with your ordering.
catalog Manually rebuilds the catalog. This can be useful if you added new files to the folders you watch but your next automatic rebuild (if enabled) is too far in the future.
version Checks if a newer version of the application is available for download.
history Clears the command history. After executing this you won't see any commands in the history window until you executed further commands.

Command history

If you have enabled the history in the settings you can bring up the history view by holding Ctrl while pressing the up or down arrow in the main window. All commands that you previously entered are displayed in the list that now shows up. By pressing the up or down arrow while holding Ctrl you can navigate through the list. If you have found the command you want to enter again, press return while still holding the Ctrl key. The selected command will now be inserted into the input field and you can execute the command.

Configure Blitzy

To configure Blitzy and all installed plugins you simply click on the settings icon in the main window. After doing so, a dialog will open where you can configure Blitzy to fit your needs. The dialog is divided into several tab items:

General

In this tab you can edit the general behaviour of Blitzy. The default settings that are installed when running Blitzy for the first time should be good enough for most of the users. However, if you want to do some fine tuning this is place to go.

Show tray icon

This option defines whether you will see the Blitzy icon in the system tray or not. The default behaviour is to show the icon.

Note: The tray icon is not needed for Blitzy to operate. It is just a reminder for you that Blitzy is running and waiting for your input.

Automatically check for updates

When this option is enabled (which is the default) Blitzy will check for new versions online every time it is started. Of course this requires an internet connection.

Close when focus is lost

When this setting is enabled the Blitzy window will disappear if it loses focus, i.e. you click outside of the window or activate a different window. The default setting is to hide the window when it loses its focus.

Close on escape

This option defines whether the Blitzy window will be closed if you press the escape key. The default behaviour is to close the window if the escape key is pressed.

Close after command execution

When this setting is enabled (which is the default) the Blitzy window will close after you execute a command.

Keep input content

When this option is enabled the last executed command will stay in the input field even if the window is closed after executing the command. The default behaviour is to clear the input field after a command was executed.

Always stay on top

When this setting is enabled (which is the default) the Blitzy window will stay on top of all other windows.

Hotkey

This setting defines which key combination will be used to show up the Blitzy window. This always has to be a combination of a system key (like Ctrl, Alt or the Windows key) and an ordinary key. The default value is Ctrl+Alt+Space

Max Items

This is the maximum amount of suggestions that show up in the suggestion window. A higher value is likely to slow down the application. A lower value is likely to be less precise. So you should test which value fits your needs best.

Folders

This tab is where you can configure which folders Blitzy should scan for files that will be listed as commands. By default your whole Start Menu will be scanned for applications.

To add a new location simply click the "Add"-button below the list of folders and then select the folder you want to add. Without further configuration all files within the selected folder will be used and the search will not include subfolders. To include subfolders (and subsubfolders and so on) select the folder you just added and check the "Recursive" checkbox.

To fine tune what files should be used from a folder you can use the "Rules" and "Excludes"

Plugins

The plugins tab is where you can disable and enable installed plugins and configure them. To enable or disable a plugin you have to select the plugin from the list of available plugins and check or uncheck the checkbox on top of the list.
When you select a plugin from the list, the available settings for the plugin will be displayed and you can change them.

About

This is the place where you can view who is responsible for all the stuff you might already love, view the details about the license, and -- and this is probably the number one reason why you would click on this tab -- you can see what version of Blitzy you are using and when it was built. If you didn't enabled the automatic update check you can check if there is a new version available.

Plugins

Blitzy's capabilities can be extended through plugins.

To install a new plugin you have to simply copy the plugin's DLL file into the 'plugin' folder of your Blitzy installation (Note: you will probably need administrator rights to do this if you have installed Blitzy to the default location). However it could be possible that there are more steps needed in order to fully install the plugin. For any further information please refer the the author of the plugin you are trying to install.

Blitzy is delivered with a couple of plugins by default, so you can decided whether you want something to be enabled or not. The plugins that are included with the Blitzy installation are listed in the following.

Calcy

Calcy is a plugin that allows you to perform calculation within Blitzy. For invoking a Calcy command you simply type:

calcy [tab] expression

And while you type the result of the expression you typed in so far will be displayed in the info field. Below is a list of functions and constants you can use in your expression.

Function Description Example
pi The value of π pi = 3.141...
e The value of e e = 2.718...
sin Calculates the sine sin(pi) = 0
cos Calculates the cosine cos(pi) = -1
tan Calculates the tangent tan(pi) = 0
sqrt Calculates the square root sqrt(4) = 2
ln Calculates the natural logarithm ln(e) = 1
log Calculates the logarithm log(100,10) = 2
asin Calculates the arcsine asin(0.1) = 0.100...
acos Calculates the arccosine acos(0.1) = 1.470...
atan Calculates the arctangent atan(0.1) = 0.099...

Weby

Weby is a fast way to search the internet using your preferred search engines. Additionally you can simply open any website from the command input. If you want to open - let's say wikipedia - you would simply type:

weby [tab] wikipedia.org

After a default installation there are some default engines already installed:

  • google – Web search using Google
  • wiki – Look up the term on Wikipedia (English version)
  • youtube – Search a video on YouTube
  • bing – Web searching using Bing
  • facebook – Search for people on Facebook
  • wolfram – Compute something using Wolfram Alpha

So if you want to search something using google you would simply type

google [tab] your search term

After executing the command your default web browser will open and the results of your search will be displayed.

You can add your own search engines or remove existing ones in Weby's settings dialog.

To add a new engine you would click on the "Add"-button and enter the name, a description (optionally) and the URL of the engine.
The name must not be empty and must not already exist.
The description may be left blank but then you won't see any further information when entering the command into the input field.
The URL is the place where Weby should take you after executing the command.
If you want to pass the search term you entered to the website simply put a {0} at the place in the URL where the search term needs to be.

Medy

Medy let's you control any media player on your system that supports media keys. Media keys are the extra keys on keyboards that allow you to play media, pause them, or walk through the playlist. Medy simply simulates a press of these keys so there are no compatibility issues with other software.

Available commands are:

  • play – Tell the player to start playing
  • pause – Tell the player to pause playing
  • next – Tell the player to go to the next track
  • prev – Tell the player to go to the previous track
  • volup – Increase volume
  • voldn – Decrease volume

So if you want to pause the currently playing song you simply execute

medy [tab] pause

Putty

Using this plugin you can directly start a Putty session from Blitzy. If you saved sessions in Putty they will be directly available inside Blitzy. If you simply want to start Putty to start a new session from there, you would simply type:

ssh

However, if you want to directly connect to a host you saved using Putty you would type:

ssh [tab] session-name

Note that instead of typing "ssh" you may also type "putty".

Reporting bugs

If you managed to crash Blitzy a window should appear that tells you that something went wrong and offers to send an error report to me. The text field will show you exactly what data will be send to me although I doubt you can do anything useful with this except checking if there are sensible information sent (which are not). Additionally you may enter an email address that can be used to contact you if further information is needed to fix your reported bug. This address won't be used for anything else than getting in touch with you if needed. If you don't want to send the report that's ok, but then you shouldn't complain that the error won't get fixed. I can't fix anything if I don't know it has to be fixed. If you sent the report I'll promise to do my best to fix the bug as soon as possible.

However if you find something that doesn't crash the program but should be fixed, changed or could be done better please please report an issue on the bug tracker. You'll find it at http://btbsoft.org/issues. Before creating a new issue, please perform a search to find out if the issue has already been reported by someone else.
